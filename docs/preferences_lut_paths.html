3D Lut Paths


The LUT Paths section allows you to view the 3D Lut paths of mrViewer.

Display

CTL_MODULE_PATH

These paths are the ones mrViewer will use to find CTL scripts. The value is taken from the environment variable CTL_MODULE_PATH.

CTL scripts

This section shows you all CTL scripts to be found in the CTL_MODULE_PATH. By following the convention, ODT_ prefixed scripts would be Output Display Transform scripts while RT_ prefixed scripts would be used for Rendering Transform scripts. Auxiliary scripts should not be prefixed.
