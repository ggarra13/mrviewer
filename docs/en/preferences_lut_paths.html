3D Lut Paths


The LUT Paths section allows you to view the 3D Lut paths of mrViewer.

Display

CTL_MODULE_PATH

These paths are the ones mrViewer will use to find CTL scripts. The value is taken from the environment variable CTL_MODULE_PATH.

CTL scripts

This section shows you all CTL scripts to be found in the CTL_MODULE_PATH. By following the convention, starting with IDT for Input Display Transforms, ODT prefixed scripts would be Output Display Transform scripts, RRT prefixed scripts would be used for Reference Rendering Transform scripts, LMT prefixes for Look Modification Transform scripts. Auxiliary scripts should not be prefixed.
