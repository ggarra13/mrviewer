Conventions Used in this Manual

To simplify the explanations in these manual, some simple conventions are used.

Clicking with any of the mouse buttons is referred with the naming
LMB
,
MMB
and
RMB
for Left Mouse Button, Middle Mouse Button and Right Mouse Button.

Keys and shortcuts are shown in bold, like:  
CTRL + H
.

Menus are referred in the following way:
Window->Preferences
, which means go to the Preferences menu in the Window menu.

Sections like the following:

> mrViewer --help

Represent commands that you would run in the Windows console or Unix shell.
