Positioning Preferences

The Positioning Preferences tab allows you to position and size the viewer on startup with no images.
Fixed Position

When active, here you can set the X and Y position of the main mrViewer window.

Fixed Size

When active, here you can set the default window size of mrViewer when it starts.


Take Current Window Values

This button, when clicked on, allows taking the current window values and filling in the Fixed Position and Fixed Size for you.
