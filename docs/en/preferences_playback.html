Playback Preferences


The playback preferences tab allows you to modify how mrViewer's playback and timeline works on start up.


Defaults

Auto Playback

When this option is turned on, as soon as mrViewer loads all the files, it will try to start playing them.

As other defaults, you can change the default speed (FPS) of the timeline as well as its looping mode. The FPS of the timeline only takes effect on sequences, not videos which already have an encoded FPS and it works only when you have Override FPS active.

Scrub Sensitivity
This option controls how scrubbing works. Smaller increments mean less travel of the mouse to skip a frame, while higher increments mean much more travel of the mouse to change the frame. Scrubbing by default is a toggle.
SHIFT
toggles it on, then you use the mouse to move left/right, and
ALT
or
CTRL
to exit the mode.


Timeline
This section allows you to toggle how the timeline displays the time. It can be in frames, seconds, time, Drop Timecode, Non-Drop Timecode.
