Loading Preferences

The Loading Preferences tab allows you to configure how the image, video and audio is loaded.


Load Sequences on Drag&Drop

When this option is on, any image that is dragged and dropped in the main mrViewer window will be treated as a sequence and the full sequence will be loaded. If this option is off, only a single frame will be loaded.

Load Sequence on File Association & Command Line

When this option is on, any image that is loaded thru a file association or through the command-line, it will attempt to load the full sequence. If it is off, only the single image will get loaded.

Auto load images as they change on disk

When this option is on, mrViewer will monitor the image on disk and will update it as soon as it sees a change in it.
