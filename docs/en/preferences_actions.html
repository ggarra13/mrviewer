Actions Preferences

The Actions Preferences tab allows you to configure what action is active by default.

Scrub

Sets scrubbing as the default action.

Move Picture

Sets move/scale the picture as the default action.

Color Area Selection

Sets the color area selection as the default action.

Pencil

Sets the pencil drawing tool as the default action.

Text

Sets the text drawing tool as the default action.

Eraser

Sets the eraser drawing tool as the default action.
