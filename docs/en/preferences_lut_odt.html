Output Device Transform (ODT)


The ODT section allows you to view the ODT settings for mrViewer. Note that unlike other sections in preferences, these values are not saved but are controlled by environment variables. This is to allow different projects to use different settings.

Default Output Device Transform (ODT)

Display

Transform

This section allows you to see the default value of the output device transform. The environment variable that controls it is:
MRV_ODT_CTL_TRANSFORM

CIE Chromaticities

This section allows you to set your monitor CIE chromaticities if known.
The environment variable that controls it is:
MRV_ODT_CTL_CHROMATICITIES

White Luminance

This section allows you to set your monitor white luminance if known.
The environment variable that controls it is:
MRV_ODT_CTL_WHITE_LUMINANCE

Surround Luminance

This section allows you to set the luminance of the environment you are in.
The environment variable that controls it is:
MRV_ODT_CTL_SURROUND_LUMINANCE

ICC Profile

This setting allows you to set the display's ICC profile if any.
