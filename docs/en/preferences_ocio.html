OCIO Preferences


The OCIO section allows you to set the OCIO settings for mrViewer. Note that unlike other sections in preferences, these values are also controlled by environment variables. This is to allow different projects to use different settings.

OCIO Preferences

OCIO Config File

The OCIO config file is the main control of the OpenColorIO pipeline. Once set, it will modify the Input Color Spaces, Devices and Views available to you from the RMB->OCIO menu.

Use OCIO

This option activates the use of OCIO in mrViewer. By default it is on, but you can turn it off to use CTL/ICC as before.

Input Color Spaces for 8-bits, 16-bits, 32-bits, float

These allow setting the input color space for each image's bit depth. The environment variables are:
MRV_OCIO_8bits_ICS,
MRV_OCIO_16bits_ICS,
MRV_OCIO_32bits_ICS,
MRV_OCIO_float_ICS.
