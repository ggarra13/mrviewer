Saving Preferences

The Saving Preferences tab allows you to configure how the reels are saved.


Use Image Path to Set Default Reel Saving Path

When this option is on, the file requester to save the reel will default to the path of the first image loaded in the current reel.

Use Relative Paths in Reel Content

When this option is on, the images saved in a reel will try to keep relative paths whenever possible. This is useful when changing platforms or OSes.
