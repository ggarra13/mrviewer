Rendering Transform (RT)


The Rendering Transform section allows you to view the RT settings for mrViewer. Note that unlike other sections in preferences, these values are not saved but are controlled by environment variables. This is to allow different projects to use different settings.

Default Rendering Transforms (RT)

Render Transforms for 8-bits, 16-bits, 32-bits, float

These allow setting the rendering transform for each image's bit depth. The environment variables are:
MRV_CTL_RT_8bits,
MRV_CTL_RT_16bits,
MRV_CTL_RT_32bits,
MRV_CTL_RT_float.


ICC Profile for 8-bits, 16-bits, 32-bits, float

These allow setting the ICC profile for each image bit depth.

The environment variables are:
MRV_ICC_RT_8bits,
MRV_ICC_RT_16bits,
MRV_ICC_RT_32bits,
MRV_ICC_RT_float.
