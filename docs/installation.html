Installing mrViewer

Installation of mrViewer can be done simply by decompressing the zip file (Windows) or the gzip tar file (Linux). This can be done with the utilities unzip or winzip (Windows) or tar in Linux.

Since v2.6.7, mrViewer is distributed with installers for each platform. The Windows installer will run and create a menu item to the start menu and, optionally, create an icon in the desktop, as well as modify the PATH variable. The Linux installers will register an entry in the DEB or RPM databases and install in:
/usr/local/mrViewer-v${VERSION}-${PLATFORM}.
They are installed using the utilities dpkg (DEB) or rpm (RPM).

The
mrViewer-v${VERSION}-${PLATFORM}/bin
directory among all things contains the file mrViewer.exe (Windows) or mrViewer.sh (Linux). Running this file will start the viewer.
