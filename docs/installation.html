Installing mrViewer

Installation of mrViewer is done simply by decompressing the zip file (Windows) or the gzip tar file (Linux). This can be done with the utilities unzip or winzip (Windows) or tar in Linux.
Decompressing the file will show up a Release directory. The mrViewer-v{VERSION}/bin directory among all things contains the file mrViewer.exe (Windows) or mrViewer.sh (Linux). Running this file will start the viewer.
