Seguimiento de Archivos

Una característica de mrViewer es su sistema de seguimiento de archivos internalizado.  Con él, cada imagen o película que cualquier persona en tu compañía vea será grabado en una base de datos generica y sera clasificada por show, secuencia y toma.

Preparando la Base de Datos

Para que mrViewer graba sus imágenes en la base de datos, es necessario que usted especifique la conexión de la base de datos asi como prepara una base de datos compatible con mrViewer.

El servidor de la base de datos puede ser cualquier máquina en su compañía y no la máquina local donde mrViewer corre, pero todas las otras máquinas necesitan poder acceder al servidor a través de una conexión TCP/IP.

Film Aura recomienda usar un servidor Linux para tu base de datos debido a su confiabilidad y su mejor performance en la red.

Actualmente, mrViewer soporta la base de datos PostgreSQL por defecto que es la mas completa de las base de datos gratis y de código abierto.  Otros drivers de base de datos puede que se agreguen en el futuro.
