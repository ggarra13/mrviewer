Instalando PostgreSQL

PostgreSQL debe ser instalado en un servidor.  Cada máquina que corre mrViewer NO necesita tener a PostgreSQL instalado, pero se comunicará con el servidor usando TCP/IP con una biblioteca que mrViewer incluye.

Instalando PostgreSQL significa bajar un archivo binario de ejecución o el código fuente de:
http://www.postgresql.org/

Si esta trabajando en una distribución de Unix distribution, puede también obtener una version precompilada de la base de datos usando su manejador de paquetes.  Por ejemplo, en una distribución de Ubuntu, usted debería ser capaz de hacer:

> apt-get install postgresql

Note: Instalación de PostgreSQL en una maquina Windows requiere el uso de una partición NTFS.

Una vez instalado, puede que quiera registrarse como administrador y crear una cuenta administrativa con permisos para acceder la base de datos.   Comunmente, esta cuenta de userio es llamada "postgres" and puede que ya haya sido creada como parte de la instalación del software.  Este usario debería tener permisos para crear bases de datos.  Para más información sobre cómo hacer esto, refierase al manual de postgreSQL.
