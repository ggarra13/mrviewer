Creando la Base de Datos

Hay dos formas de crear tu base de datos.  Una es usando SQL basico que es específica a cada base de datos.

La otra forma, más poderosa, es usando rails' migrations con el proyecto de rails provisto con mrViewer.  De cualquier forma, usted tendrá que crear un usuario administrativo en la base de datos que tendra una clave y permisos para crear o descartar tablas en la base de datos.

Este usuario debería llamarse mrviewer o tendra que ajustar el archivo en rails/assets2d/config/database.yml.

Una vez que eso esté hecho, deberá correr de una ventana como usuario root:

$ apt-get install ruby1.9.1 rubygems # or cualquier package manager que su distro de Linux use
$ gem install rails
$ gem install rake

Ahora, desde el directory principal de rails/assets2d:

> rake db:schema:load

Eso creará la base de datos llamada asset2d_development. Para crear la base de datos de producción, use:

> export RAILS_ENV=production # for bash, use set for mswin console

> rake db:schema:load

The verdadera base de datos es assets2d_production y el programa de mrViewer la usará, pero las otras se ofrecen como pruebas.
