Problemas en Windows

mrViewer no muestra sus opciones de comando.

Siempre que arranco mrViewer, no deja información en la consola.

Respuesta:
Este es un problema con Windows. Para ver las opciones tipee:
mrViewer -h

Las opciones aparecerán en una nueva consola. To ver el resto de los mensajes que mrViewer genera, vaya al menú
Ventana->Mensajes
