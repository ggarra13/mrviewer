Variables de Entorno

Al comenzar y durante diferentes operaciones, mrViewer lee varias variables de entorno que te permiten customizar su operación.  Aquí hay una lista de ellas:

Variable Explicación Valor por Defecto
CTL_MODULE_PATH Una línea de texto con caminos separados por dos puntos (Unix) o semicoma (Windows) donde mrViewer encontrará .ctl scripts. Si no está seteado, el subdirectorio ctl de mrViewer
MRV_ODT_CTL_DISPLAY_TRANSFORM El script de CTL usado para transformar imágenes del espacio CIExy space para mostrar en tu pantalla Ninguno.
MRV_ODT_ICC_PROFILE El perfil ICC atado al monitor para simular el medio de salida. Ninguno.
MRV_ODT_CTL_DISPLAY_CHROMATICITIES Cromaticidades de tu display Rec. ITU-R BT.709 chromaticidades
MRV_ODT_CTL_DISPLAY_WHITE_LUMINANCE Luminancia de Blanco de tu display 120 candelas por metro cuadrado
MRV_ODT_CTL_DISPLAY_SURROUND_LUMINANCE Luminancia del entorno de alrededor 12 candelas por metro cuadrado
MRV_CTL_RT_8bit Una script CTL por defecto usada para transformar imágenes de 8-bits de RGB al espacio CIExy Ninguna.
MRV_CTL_RT_16bit Una script CTL por defecto usada para transformar imágenes de 16-bits de RGB al espacio CIExy Ninguna.
MRV_CTL_RT_32bit Una script CTL por defecto usada para transformar imágenes de 32-bits de RGB al espacio CIExy Ninguna.
MRV_CTL_RT_float Una script CTL por defecto usada para transformar imágenes de coma flotante o media coma flotante de RGB al espacio CIExy RRT.a1.0.0
MRV_ICC_RT_8bit Un perfil ICC por defecto usado para transformar imágenes de 8-bits de RGB al espacio CIExy Ninguno.
MRV_ICC_RT_16bit Un perfil ICC por defecto usado para transformar imágenes de 16-bits de RGB al espacio CIExy Ninguno.
MRV_ICC_RT_32bit Un perfil ICC por defecto usado para transformar imágenes de 32-bits de RGB al espacio CIExy Ninguno.
MRV_ICC_RT_float Un perfil ICC por defecto usado para transformar imágenes de coma flotante o media coma flotante de RGB al espacio CIExy Ninguno.
MRV_HOST Maquina donde la base de datos de mrViewer reside. localhost
MRV_PORT Puerto TCP/IP Port para usar la conexión a la base de datos Usa por defecto para la base de datos.
MRV_USER Usuario usado para loguearse a la base de datos de mrViewer mrviewer
MRV_PASSWORD Clave usada para loguearse a la base de datos de mrViewer mrviewer
MRV_DATABASE Nombre de la base de datos de mrViewer assets2d_production
SHOW Si está seteado, cualquier imagen vista sera grabada en la base de datos como perteneciente a dicho show. Indefinida.
SEQ Si está seteada, cualquier imagen vista sera grabada en la base de datos como perteneciente a dicha secuencia. Indefinida.
SHOT Si está seteada, cualquier imagen vista sera grabada en la base de datos como perteneciente a dicha toma. Indefinida.
ICC_PROFILES Directorio donde los perfiles ICC e ICM residen. %WINDIR%/SYSTEM32/spool/drivers/color en Windows y /usr/share/color/icc en Unix
HOME Directorio Home para el usuario donde las preferencias de mrViewer serán grabadas. $HOME en Unix y %HOMEDIR% en Windows
MRV_SHADER_PATH Directorio donde los shaders de mrViewer residen. Indefinido, mrViewer usa el directorio shaders/ o un shader prefijado.
