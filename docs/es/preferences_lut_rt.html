Transformación De Rendering (RT)


La sección de Transformación De Rendering section te permite ver los seteos de RT para mrViewer. Note que contrariamente a otras secciones en las preferencias, estos valores no son grabados, sino son controlados por variables de entorno. Este es para permitir que diferentes proyectos usen diferentes seteos.

Default Transformación De Renderings (RT)

Transformaciones de Render para 8-bits, 16-bits, 32-bits, float

Estos permiten setear la Transformación de Render para cada imagen segun su profundidad. Las variables de entorno son:
MRV_CTL_RT_8bits,
MRV_CTL_RT_16bits,
MRV_CTL_RT_32bits,
MRV_CTL_RT_float.


Perfil ICC for 8-bits, 16-bits, 32-bits, float

Estos permiten setear el perfil ICC para cada imagen segun su profundidad.
Las variables de entorno son:
MRV_ICC_RT_8bits,
MRV_ICC_RT_16bits,
MRV_ICC_RT_32bits,
MRV_ICC_RT_float.
