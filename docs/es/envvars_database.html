Seteando las Variables de Entorno

Ahora, para que mrViewer pueda acceder a la base de datos, un set de variables de entorno deben ser seteadas.  mrViewer leerá estas variables al comenzar e intentará communicarse con la base de datos.  

Para Windows, estas variables deberían ser seteados por un administrator or ejecutando un archivo .bat que arranque mrViewer.  

Bajo Unix systems, mrViewer ya viene con una fila de arranque llamada 'mrViewer.sh' que setea estas variables a su valor por defecto.  Debería modificar este archivo de bash para ajustarlo a sus necesidades.

Los valores de las variables por defecto son:

MRV_DRIVERpostgresql
MRV_HOST127.0.0.1
MRV_USERmrviewer
MRV_PASSWORDmrviewer
MRV_DATABASEassets2d_production
MRV_PORT5432
