Preferencias del Buscador de Archivos

Esta pantalla te permite configurar cómo interactuás con el buscador de archivos.

Un Click para Navegar Carpetas

Cuando esta opción esta activa, un sólo click te permitirá navegar a través de los directorios. Cuando esta inactiva, tendrás que cliquear dos veces en la carpeta para navegar en ella.

Usar el Selector de Archivos Nativo en Windows

Aquí podes elegir usar el buscador de archivos nativo de Windows en vez del que viene con mrViewer.
