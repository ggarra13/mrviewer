Accediendo el Proyecto
de Ruby on Rails en tu Navegador

Dependiendo si tenes tu setup de rails para trabajar en modo desarrollo o modo de producción, la localidad de la página web para la base de datos variará.
Durante el desarrollo, puedes encontrar el proyecto de rails en el puerto 3000 de la máquina.  Una vez que el proyecto de rails es desplegado, lo encontrará en la dirección IP o dirección de web que usted haya decidido instalarlo.

Para el propósito de este tutorial, asumiremos que está trabajando en modo de desarrollo.
Vaya a una ventana de shell o consola y desde el directorio de rails/assets2d de mrViewer tipee:

> rails server

Eso imprimirá un número de advertencias en varios módulos. Ignorelos porque son inofensivos. Una vez realizado esto, si usted está actualmente logueado en la máquina donde reside la base de datos, esto usualmente significa que apuntando tu navegador a:

http://localhost:3000/images

Una vez que llegue allí, podrá navegar a las secciones de administración, shows, secuencias, tomas, audios, imágenenes, videos y categorías.

Entonces, para acceder a la sección de videos, usted iría a:

http://localhost:3000/videos
