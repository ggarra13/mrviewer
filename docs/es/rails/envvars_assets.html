Variables de Entorno para Assets

Cuando mrViewer muestra una imagen, película o audio, puede asentarlo en la base de datos. A veces es util atar una toma, secuencia o show a la imagen que está siendo catalogada. mrViewer depende de algunas variables de entorno útiles para ello.

La variable de entorno SHOW determina a que show pertenece la imagen. Debería ser un nombre corto como 'hulk'.

La variable de entorno SHOT determina la toma a cual la imagen pertenece. Deberia ser un texto de 2 a 3 caracteres seguido por un número. Por ejemplo: da023, ep123.

La variable de entorno opcional SEQ determina la secuencia a la que pertenece la imagen. Si no es seteado, ésta es adivinada de las primeras letras de la toma (SHOT). Como con el show, el valor de la misma debe ser un nombre corto, como el prefijo de la toma.

Finalmente, la variable de entorno USER o USERNAME es usada para catalogar quien vió la imagen por vez primera.
