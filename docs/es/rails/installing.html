Instalando el Servidor de Rails

Instalación de rails puede hacerse con el manejador de paquetes en Linux o a través del setup.exe de ruby on rails de windows.

Una vez instalado, debes descargar el plugin active scaffold. Para ello:

$ sudo gem install active_scaffold

or a través de rails directamente con:

$ sudo rails plugin install http:active_scaffold_address

También necesitas instalar el plugin de ImageMagick llamado RMagick:

$ sudo gem install rmagick
