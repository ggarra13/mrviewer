Instalando mrViewer

La instalación de mrViewer puede ser hecha simplemente descomprimiendo el archivo zip (Windows) o el archivo gzip tar (Linux). Esto puede ser hecho con las utilidades unzip o winzip (Windows) o tar en Linux.

$ unzip mrViewer-v${VERSION}-Windows-32.zip


$ tar -xf mrViewer-v${VERSION}-Linux-64.tgz


Desde la version v2.6.7, mrViewer es distribuído con instaladores para cada platforma.

Algunos antivirus se quejan que el instalador de Windows puede ser nocivo. No hay que prestarle atención. Si duda del instalador, remítase al archivo zip. El instalador de Windows correrá y puede crear un item de menu en el menu de Comienzo o, opcionalmente, crear un ícono en el Escritorio, así como modificar la variable PATH de entorno. Notese que si usted modifica la variable PATH, puede que no pueda correr la version de 32bits de mrViewer junto a la version de 64bits. También puede que cree problemas con otros programas que usan algunas de las bibliotecas de mrViewer.

Los instaladores Linux necesitan permiso de administrador, como:

sudo dpkg -i mrViewer-v${VERSION}-${PLATFORM}.deb

Ellos registrarán una entrada en las bases de datos DEB or RPM respectivamente y se instalarán en:

/usr/local/mrViewer-v${VERSION}-${PLATFORM}

Las utilidades dpkg (DEB) or rpm (RPM) adicionalmente crearán uniones simbólicas (symlinks) en /usr/bin/ e instalaran el ícono en /usr/share/applications que hará que aparezca en los menúes (Graphics). El ícono también será copiado al Escritorio del usuario.

El directorio
mrViewer-v${VERSION}-${PLATFORM}/bin
entre todas las cosas contiene el archivo mrViewer.exe (Windows) o mrViewer.sh (Linux). Ejecutar ese archivo o cualquiera de los íconos iniciará el visor.


Finalmente, los usuarios de Windows tienen la posibilidad de usar chocolatey para instalar/hacer un upgrade de mrViewer. Chocolatey es un instalador remoto similar a apt-get o yum en Linux. Para usarlo, primero debe bajar e instalar chocolatey de chocolatey.org (siga las instrucciones de su sitio web --inglés solamente). Luego, un comando llamado 'choco' estará disponible. Entonces podrá ejecutar:

choco install mrviewer


para instalar mrViewer por vez primera y:

choco upgrade mrviewer


para hacer un upgrade de mrViewer a la última versión. Note que el instalador de chocolatey creará un ícono en el Escritorio, pero no asociará ninguna extensión de archivo con mrViewer.


En las Macs, mrViewer es distribuído como un archivo .dmg. Podes descomprimirlo si cliqueas en él dos veces o ejecutas la utilidad de comando hdiutil desde la terminal, como:

hdiutil attach mrViewer-${VERSION}-Darwin-64.dmg

Se te presentará la licencia de mrViewer que deberás aceptar y luego el instalador creará un volúmen para mrViewer. Abriendo ese icono revelará la aplicación mrViewer y un enlace simbólico a la carpeta de Aplicaciones. Puedes correr mrViewer directamente desde la ventana del Finder o instalarlo deslizando el ícono a la carpeta de Aplicaciones (el método sugerido).
Note que la primera vez que ejecute mrViewer, deberá correrlo con
CTRL+BIR->Abrir
para darle permisos ya que el ejecutable de mrViewer no está notariado por Apple. Si eso no funciona, otra opcion es correr el reproductor dos veces en sucesióm lo cual debería preguntarte con una solicitud para Abrir el archivo.
