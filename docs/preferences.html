Preferences Window


You can bring up the Preferences Window by doing any of:
  • Opening up the window from the main toolbar's
    W
    button.
  • By using
    RMB->Window->Preferences.

The Preferences Window allows you to configure some current and default preferences of how mrViewer works when started.  Each section is divided in the menu to the left, so you should click on each row to get access to each section.

The current sections are:

  • User Interface
  • View
  • Playback
  • Loading
  • Caches
  • Video
  • Audio
  • 3D LUT
  • LUT Paths
  • LUT ODT
  • LUT RT
